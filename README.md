# 📖 About

This repository documents my journey to becoming a better Flutter Software Engineer.

The goal is **not** to collect notes.

The goal is to deeply understand how software works, document every important concept, and build engineering thinking through structured learning.

Every lesson is written after understanding the concept—not by copying tutorials.

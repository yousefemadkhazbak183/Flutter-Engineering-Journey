# 001 - Why Do We Need Programming?

## 🎯 Goal

Understand why programming languages exist and how humans communicate with computers.

---

# The Problem

A computer does not understand human languages such as:

- Dart
- Java
- Python
- C++

It only understands **Machine Code (Binary)**.

Example:

```
010101001010101010101...
```

---

# Why Does the Computer Understand Only 0 and 1?

Inside the CPU there are billions of electronic switches called **Transistors**.

Each transistor has only two states:

- ON
- OFF

We represent them as:

```
0 = OFF
1 = ON
```

This is why computers use the Binary System.

---

# The Challenge

Imagine writing an application using only:

```
101010101010101...
```

It would be extremely difficult for humans.

So engineers needed a better solution.

---

# The Solution

Humans created Programming Languages.

Instead of writing:

```
1010101010...
```

We can write:

```dart
print("Hello");
```

This is much easier for humans.

---

# But There Is a Problem

The CPU still doesn't understand:

```dart
print("Hello");
```

So we need another program.

---

# The Translator

A special program converts our code into Machine Code.

Example:

```
Dart Code

↓

Compiler

↓

Machine Code

↓

CPU
```

The CPU never sees Dart code.

It only receives Machine Code.

---

# How Did It Start?

The first programmers wrote directly in Machine Code.

Then Assembly Language was invented.

```
Machine Code

↓

Assembly

↓

High-Level Languages

↓

Modern Languages (Dart, Kotlin, Swift...)
```

Each step made programming easier.

---

# Key Concepts

- Binary
- Machine Code
- CPU
- Transistor
- Programming Language
- Assembly Language
- Compiler

---

# What I Learned

- Computers understand only Machine Code.
- Programming languages were created for humans.
- A Compiler translates code into Machine Code.
- Modern languages exist because they improve developer productivity.

---

# Summary

Programming languages were invented to make communication between humans and computers easier.

The computer understands only Machine Code, so a Compiler translates our code into a language the CPU can execute.

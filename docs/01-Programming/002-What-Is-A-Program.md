# 002 - What Is a Program?

## 🎯 Goal

Understand what a program is and how a computer executes it.

---

## 💡 The Problem

Imagine you have a robot.

If you simply say:

> Make me a cup of coffee.

The robot won't know what to do.

Instead, you must provide clear instructions:

1. Go to the kitchen.
2. Take a cup.
3. Add coffee.
4. Add sugar.
5. Pour hot water.
6. Stir.
7. Serve.

---

## The Idea

A computer works exactly the same way.

It doesn't understand your goal.

It only understands instructions.

---

## Definition

A **Program** is:

> A sequence of instructions executed by a computer to accomplish a specific task.

---

## Example

```dart
void main() {
  print("Hello");
}
```

Execution:

```
Start

↓

Print "Hello"

↓

End
```

---

## Program Execution

A computer executes instructions sequentially:

1. Read the next instruction.
2. Execute it.
3. Move to the following instruction.
4. Repeat until the program finishes.

---


## 🔑 Key Concepts

- Program
- Instruction
- Execution
- Sequential Execution
- Start
- End

---

## 📝 Summary

A program is a sequence of instructions.

The computer executes those instructions one by one until the task is completed.
